# Roadmap

## Target

Ghostshift is tracking toward `OSS Platform v1` on **2026-06-12**.

The release train is:

- `v0.7` Integration Foundation
- `v0.8` Provenance And Team Workflow
- `v0.9` Self-Host Preview
- `v1.0` OSS Platform Launch Hardening

## v0.7 Integration Foundation

Status: shipped in this scaffold.

- stable plugin API
- official `git` adapter
- official `shell` adapter
- patch-aware export payload
- derived reports in export output
- adapter authoring docs
- export format docs

## v0.8 Provenance And Team Workflow

Status: shipped in this scaffold.

- decision-linked blame
- decision and patch linkage in export
- compare reports with stronger provenance summaries
- more shareable CLI reports
- backward-compatible schema discipline

## v0.9 Self-Host Preview

Status: shipped in this scaffold.

- minimal server preview
- minimal web UI
- session list/detail views
- explain and compare views
- blame lookup view
- local-first import/sync path from CLI data

## v1.0 OSS Platform Launch Hardening

Status: shipped in this scaffold.

- GitHub PR summary flow
- release packaging and install ergonomics
- stable quickstart and architecture docs
- example repos and workflows
- migration and compatibility notes
- launch checklist and smoke tests
